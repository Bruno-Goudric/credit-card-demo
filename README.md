# Credit-card-demo
> Está é uma aplicação simples de estudo desenvolvida com ReactJs onde a mesma executa a animação na hora de fazer o pagamento.

[![Build Status][travis-image]][travis-url]

## Instalação

```sh
npm install
```

## Execut

```sh
npm start
```
